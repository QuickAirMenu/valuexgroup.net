# VALUE X GROUP — Official Website

**valuexgroup.net** — bilingual corporate website for Value X Group for Business Development, Kingdom of Saudi Arabia.

Developed & Designed by **[Air Menu](https://airmenu.net)**

## Stack

- HTML5 / CSS3 / Vanilla JS
- Font Awesome 6.5.1
- Google Fonts (Cairo, Tajawal, Cormorant Garamond, Inter)
- Graphik Arabic (Local Font)
- Hostinger Shared Hosting

## Structure

```
ar/index.html      ← Arabic homepage (RTL)
ar/privacy.html    ← Arabic privacy policy
ar/terms.html      ← Arabic terms & conditions
en/index.html      ← English homepage (LTR)
en/privacy.html    ← English privacy policy
en/terms.html      ← English terms & conditions
assets/
  style/style.css  ← Shared stylesheet
  js/app.js        ← Shared JavaScript
  img/logo/        ← Brand logos (white, gold, black)
  fonts/           ← Graphik Arabic SemiBold
sitemap.xml        ← XML Sitemap
robots.txt         ← Robots.txt
```
